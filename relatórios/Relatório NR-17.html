Objetivo do Curso

Capacitar operadores de teleatendimento para a compreensão e aplicação prática da Norma Regulamentadora 17 (Anexo II), promovendo a autogestão do conforto ergonômico, prevenção de doenças ocupacionais e melhoria da saúde no ambiente de trabalho.

1. Arquitetura Pedagógica

O curso "NR-17 Aplicada" foi desenhado sob uma perspectiva andragógica e prática, focando não apenas na transmissão de conceitos normativos, mas na efetiva mudança de comportamento e conscientização sobre saúde ocupacional.

1.1. Fundamentos Instrucionais

Taxonomia de Bloom Revisada: O curso estrutura-se para levar o aluno além da memorização:

  • Lembrar: Identificar riscos e requisitos ergonômicos básicos.
  • Entender: Compreender a relação entre postura e saúde a longo prazo.
  • Aplicar: Ajustar corretamente seu próprio posto de trabalho (cadeira, monitor, iluminação) em tempo real.
  • Analisar: Distinguir cenários de risco versus cenários seguros em simulações.

Andragogia (Modelo de Knowles): O conteúdo respeita a autonomia do adulto:

  • Necessidade de Saber: Cada módulo explica o "porquê" do aprendizado (ex: reduzir dor).
  • Autoconceito: O operador é tratado como "Gestor do Próprio Conforto", responsável ativo.
  • Orientação para Problemas: O foco está em resolver dores reais e desconfortos do dia a dia.

2.3. Acessibilidade e Inclusão

O projeto foi desenvolvido com base nas diretrizes WCAG 2.1 AA, implementando um conjunto de recursos integrados para facilitar o acesso ao conteúdo.

Internacionalização Completa

Conteúdo integralmente traduzido para Português, Inglês, Espanhol e Francês. A alternância de idioma é instantânea e persiste entre sessões.

Leitura Guiada Avançada

Utiliza a Web Speech API para narração automática. Inclui controles de velocidade de leitura (0.75x a 2x) e seleção de voz (masculina/feminina) diretamente nas configurações.

Adaptação Visual

Ferramentas de conforto visual incluem:

  • Alto Contraste: Modo noturno.
  • Regulação de Fonte: Ajuste granular do tamanho do texto.
  • Modo Dislexia: Fonte OpenDyslexic para facilitar a leitura.
VLibras Integrado

Widget de tradução automática para Língua Brasileira de Sinais (Libras), garantindo acessibilidade para pessoas surdas ou com deficiência auditiva.

Tutorial Interativo

Sistema de Onboarding Guiado que ensina o usuário a navegar pela interface e utilizar os recursos de acessibilidade no primeiro acesso.

O curso implementa tecnologia PWA (Progressive Web App) para garantir o funcionamento contínuo mesmo em ambientes com internet instável, como lojas e quiosques remotos. Todo o conteúdo é armazenado localmente após o primeiro acesso.

Recurso Benefício Operacional

Robustez em Lojas Ideal para colaboradores em pontas de varejo com conectividade limitada ou compartilhada.

6. Manutenção e Ferramentas

6.1. Suite de Automação e Ferramentas

Para garantir a manutenção e escalabilidade do curso, foram desenvolvidas ferramentas dedicadas de automação em Python:

  • Build Seguro (build_secure.py): Gera a versão de distribuição (dist_scorm_secure), aplicando minificação, ofuscação e limpeza de comentários para produção.
  • Auditoria de Chaves (verify_locales.py): Varre recursivamente todos os arquivos JSON para identificar chaves faltantes ou tipos de dados inconsistentes entre o idioma mestre (PT) e os demais.
  • Garantia de Estrutura (ensure_locales.py): Cria automaticamente a árvore de arquivos para novos idiomas, replicando a estrutura do idioma base para evitar erros de "File Not Found".
  • Extração de Textos (extract_i18n.py): Automatiza a extração de strings hardcoded de novos arquivos HTML para os JSONs de tradução.
  • Limpeza de HTML (prune_html_text.py): Remove textos originais dos arquivos HTML após a extração, garantindo que todo o conteúdo textual seja servido via atributos `data-i18n`.
