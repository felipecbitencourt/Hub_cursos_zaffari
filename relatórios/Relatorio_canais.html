Nota de Escopo

Por se tratar de um microlearning de acompanhamento do curso "Por Dentro do SAC", muitas das bases estruturais se mantêm as mesmas. Este relatório propõe-se a apresentar apenas as diferenças específicas presentes neste curso e as melhorias implementadas. A versão publicada do micro está disponível em: https://dudatess.github.io/sac-micro-2/ (Canais de Comunicação SAC — demonstração GitHub Pages).

Objetivo do Curso

Capacitar a equipe de atendimento sobre os diversos canais de comunicação do SAC, suas especificidades, fluxos de atendimento e ferramentas (Salesforce), garantindo um serviço padronizado e de excelência em todos os pontos de contato.

1. Arquitetura Pedagógica

O curso "Canais de Comunicação SAC" é um módulo focado e direto, desenhado para apresentar rapidamente o ecossistema de atendimento ao qual o colaborador estará inserido.

1.1. Estrutura Curricular

O curso está organizado em 3 seções principais:

🎯 Início

  • Antes de Avançar (Intro e Configurações)

📱 Módulo 1: Canais de Comunicação

Apresentação detalhada de cada ponto de contato:

  • 📍 Canais de Comunicação (Visão Geral)
  • 💻 Nosso Sistema (Salesforce) Novo
  • 📱 Redes Sociais
  • 🏪 Lojas
  • 👤 Atendimento Pessoal
  • 🌐 Site
  • 📲 App
  • ✉️ E-mail
  • 🏆 Conclusão

🎮 Exercícios

Atividades práticas para fixação:

  • Monte o Fluxo (Drag and Drop)

1.2. Metodologias de Ensino

Metodologia Aplicação no Curso
Gamificação Uso de mascote, feedbacks visuais, barras de progresso e sistema de conquistas.
Microlearning Conteúdo dividido em pílulas de conhecimento por canal, facilitando a consulta rápida.
Aprendizagem Visual Uso intensivo de ícones e layouts em grid para facilitar a memorização dos canais.

2. Design e Interface do Usuário

O projeto passou por um rebranding recente para alinhar-se a uma identidade visual mais vibrante e energética.

2.1. Identidade Visual (Novo Padrão)

Elemento Especificação Atualização
Cores Primárias Amber/Orange (#FFA000) Substituiu o antigo vermelho padrão
Design System Modern Premium Glassmorphism, sombras suaves e bordas arredondadas
Interatividade Animações CSS Feedback visual em hover e clique

3. Arquitetura Técnica

Além da compatibilidade SCORM, o curso foi atualizado com capacidades de Progressive Web App (PWA).

3.1. Recursos Avançados

Recurso Tecnologia Benefício
PWA (Offline) Service Worker + Manifest Permite instalação como App e funcionamento sem internet após o primeiro acesso.
Acessibilidade Libras VLibras Widget Tradução automática de conteúdo para Língua Brasileira de Sinais.
Internacionalização i18n.js (JSON Modular) Suporte nativo e instantâneo a 4 idiomas sem recarregar a página.

4. Manutenção e Ferramentas

4.1. Ferramentas de Build

Para garantir a otimização e distribuição do curso, foi desenvolvido um script de automação em Node.js (build.js).

Funcionalidade Descrição
Minificação Redução automática de arquivos HTML, CSS e JS para melhor performance.
Limpeza Remoção automática da pasta dist/ anterior para evitar conflitos.
Distribuição Geração de estrutura pronta para pacote SCORM ou deploy web.

4.2. Internacionalização (i18n)

O gerenciamento de idiomas é feito via JavaScript (assets/js/i18n.js) no lado do cliente, permitindo:

  • Carregamento dinâmico de arquivos JSON de tradução.
  • Fallback automático para Português (pt-BR) caso a tradução falhe.
  • Troca de idioma instantânea sem recarregamento da página.
