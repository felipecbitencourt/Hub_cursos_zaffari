# Anotações e Changelog de Conversas

Este documento centraliza registros de conversas para atualização contínua de atividades do projeto.

## Como atualizar

- Adicione uma nova seção para cada interação relevante.
- Mantenha os campos padrão: data, horário, envolvidos, projeto e atividade.
- Em "Resumo" descreva em 1-3 linhas o que foi tratado.
- Em "Próxima ação" descreva o próximo passo objetivo.

## Modelo de entrada

```md
### Registro YYYY-MM-DD-HHMM
- **Data:** DD/MM/AAAA
- **Horário:** HH:MM
- **Envolvidos:** Nome 1, Nome 2
- **Projeto:** Nome do projeto
- **Atividade:** Tipo de atividade
- **Origem:** chat.txt
- **Resumo:** Texto curto com os principais pontos.
- **Próxima ação:** Próximo passo acordado.
```
